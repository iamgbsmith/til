# List Databases And Tables Using PSQL

__Category: Postgres__

`psql` is a command line application for working with Postgres databases.

Launch psql:

```shell
psql
```

List databases:

```shell
\l
```

Use a database:

```shell
\c mydatabase
```

List database tables:

```shell
\dt
```

Exit psql:

```shell
\q
```
