# Connect A Menu Item To Code

__Category: Xcode__

You can connect a menu item in a storyboard to a method in a Swift or Cocoa class by creating a referencing outlet.

Switch the editor to a split view to show the code for the class file and storyboard side-by-side.

Click on the menu item the storyboard, hold down the `Ctrl` key and drag the item onto the class file for the referencing outlet, releasing the key and mouse when done.
